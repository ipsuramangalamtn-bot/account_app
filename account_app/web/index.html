🔥 Firebase Setup Guide

5 minutes · Free · No server needed

1

Create a Firebase Project

Go to console.firebase.google.com → Add project → name it (e.g. account-portal) → Create project.

2

Enable Firestore Database

Left sidebar → Firestore Database → Create database → Start in test mode → Choose region → Enable.

3

Get Your Web App Config

Project Settings (⚙️) → Your apps → </> Web icon → Register. Copy the config object that looks like:

const firebaseConfig = { apiKey: "AIzaSyXXXXXXX", authDomain: "account-portal.firebaseapp.com", projectId: "account-portal", storageBucket: "account-portal.appspot.com", messagingSenderId: "123456789", appId: "1:123:web:xxxxxxxx" };
4

Paste into portal.html

Open portal.html in Notepad / VS Code. Find const FIREBASE_CONFIG = { near the top. Replace all placeholder values with your actual values. Save and reload.

✅ The yellow banner disappears and status shows 🟢 Live. Now upload your Office Master Excel in Admin — it saves to Firebase and all users get it instantly on page load!
5

Share with Your Team

Drag portal.html to netlify.com/drop for a free public URL everyone can access on mobile or desktop.
